# Butay

Repositorio del proyecto Butay: marca de streetwear emocional y su web
oficial.

## Estructura

```
/
├── docs/          → documentación completa del proyecto (marca, producto, metodología, entrega técnica)
├── butay-web/     → código fuente de la web (Next.js + TypeScript + Tailwind)
├── .gitignore
└── README.md
```

## Para desarrolladores / IA que retomen el proyecto

Antes de escribir una sola línea de código, lee en este orden:

1. [`docs/CONTEXT.md`](docs/CONTEXT.md) — estado actual del proyecto.
2. [`docs/INDEX.md`](docs/INDEX.md) — mapa de toda la documentación.
3. [`docs/CLAUDE_CODE.md`](docs/CLAUDE_CODE.md) — puerta de entrada técnica.
4. [`docs/WEB_HANDOFF.md`](docs/WEB_HANDOFF.md) — resumen técnico de marca y producto.
5. [`docs/FRONTEND_ARCHITECTURE.md`](docs/FRONTEND_ARCHITECTURE.md) — arquitectura técnica detallada.
6. [`docs/DEVELOPMENT_ROADMAP.md`](docs/DEVELOPMENT_ROADMAP.md) — fase actual del desarrollo.

Ver `docs/CLAUDE_CODE.md` (apartado 2) para el detalle completo del
orden de lectura obligatorio.

## Estado
